# PROJECT_BRIEF

## 프로젝트 한 줄 설명

`괴담기록국`은 요원 팀을 편성해 도시괴담의 규칙을 조사하고, 단서를 근거로 괴이를 안정화·회수하는 PC용 현대 오컬트 수사 어드벤처다.

## 장르와 플랫폼

- 현대 오컬트 미스터리, 텍스트 노벨형 조사 어드벤처, 요원 팀·캠페인 운영
- Godot 4.7 stable / GDScript
- PC / Steam, 16:9, 마우스·키보드

## 플레이어 역할

플레이어는 고정 주인공이 아니라 괴담기록국 요원 팀의 판단과 일정을 맡는다. 현재 반일의 활동을 계획하고, 현장에서 상황과 단서를 읽고, 회수 담당과 대응을 선택한다.

## 핵심 경험

> 괴담은 죽이는 게 아니라, 규칙을 밝혀 봉인하는 것이다.

- 단서와 힌트를 구분하고 근거를 축적한다.
- 요원의 능력과 짧은 반응으로 함께 수사하는 감각을 얻는다.
- 실패에서도 다음 대응에 쓸 패턴을 배운다.
- 사건 보고서와 DB에 자신의 판단을 기록한다.

## 핵심 루프

```text
반일 계획 → 조사·휴식·의뢰 → 현장 선택 → 단서·판정
→ 괴이 안정화·회수 → 결과 확인 → 보고서·DB → 다음 반일
```

## 차별점

- 괴담을 처치 대상이 아니라 규칙을 이해해 회수하는 대상으로 다룬다.
- 실패가 진행 차단보다 학습 기록으로 이어진다.
- 요원을 수치 카드가 아니라 수사 파트너로 표현한다.
- 장비·연구·시장은 정답 구매가 아닌 위험 관리 수단이다.
- 짧은 사건 루프와 최대 10일 캠페인의 장기 선택을 연결한다.

## 제외할 것

- HP·공격·처치 중심 전통 RPG 전투, 연애 호감도, 영구 사망
- 단서 없는 반전, 찍기식 추리, 정답 판매형 시장
- 모바일 세로 화면 기본 최적화, 요청되지 않은 온라인·결제 시스템
