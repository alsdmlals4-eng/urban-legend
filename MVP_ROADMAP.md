## MVP-017 작업 범위

MVP-017은 사건 보고서와 전체 흐름 점검을 묶는다.

### ChatGPT 작업 범위

- Issue #17 작성
- 프로젝트 기준 문서 생성
- MVP 상태 점검 문서 생성
- 전체 흐름 HTML 대시보드 생성
- Codex Goal 작성

### Codex 작업 범위

Codex는 Issue #17과 `docs/CODEX_GOAL_MVP_017.md`를 기준으로 구현한다.

예상 구현:

- `GameState.get_case_report_summary()` 추가
- 결과 화면에 사건 보고서 영역 추가
- 요원 신뢰도/요원 이벤트 결산 표시
- 수집 단서, 힌트 수, 미니게임 결과, 회수 결과, 연구 보상 표시
- README 최신화

## 제외 범위

MVP-017에서 제외한다.

- 장기 요원 개인 루트
- 요원별 전용 엔딩
- 새 미니게임
- 새 전투/회수 시스템
- 복잡한 재판식 추리
- 플로우차트 편집기
- 기존 저장 데이터 파괴
