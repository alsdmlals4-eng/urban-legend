# Project Context

## 한 줄 정의

`괴이 기록국`은 인간의 마음에서 되살아나는 괴이의 규칙을 조사하고, 현재 출현을 안정화한 뒤 다음 피해를 막을 괴이 매뉴얼을 남기는 PC용 현대 오컬트 수사 어드벤처다.

## 현재 구현 기준

- MVP-043 / Ver 4.1 / 저장 스키마 mvp-039 (`mvp-038` 이관 지원)
- 최대 10일, 오전·오후 순차 반일 캠페인
- 구현 사건: 저승역, 비 오는 골목의 빨간 우산, 폐주파수 방송국
- 폐주파수 방송국은 위험도 60에서 Day 8 오전에 자연 발견 뒤 준비 화면에 노출되며, 세 사건 QA를 통과했다.
- HQ 일상 3장은 관련 사건이 `lead`·미해결일 때만 열리고 일정·조사·위험도·세력 의뢰를 소모하지 않는다.
- 세력: 소문시장, 마도회, 퇴마사 계보
- 요원 능력: 제압·분석·방호·치료·교감
- 권나래는 고정 주인공이며 초기 요원은 권나래·윤서하·오현·강이준·한유리 5명이다. 서포트는 나머지 네 명 중 최대 2명이며 저장 스키마와 사건 데이터는 바꾸지 않는다.

## 사용자 승인 콘텐츠 방향 v0.9

`docs/CONTENT_DIRECTION_V09.md`는 다음 콘텐츠·세계관 조정의 승인 설계다. 아직 코드·데이터·GDD 통합 완료를 뜻하지 않는다.

- 공식 기관명: **괴이 기록국**
- 사건 완료 표기: **안정화 상태**
- 실패 기록: **위험 사례**
- 회수 대상: **잔향**
- 최종 기록 보상: **괴이 매뉴얼 작성·갱신**
- 괴이는 인간의 마음·기억·소문에서 발생하며 완전히 죽지 않는다.
- 기관은 현상을 조사·안정화하고 재출현 대응 절차를 기록해 인류를 수호한다.
- 첫 자유 반일 계획보다 짧은 세계관 소개와 저승역 튜토리얼을 먼저 제공한다.
- 저승역은 막차를 놓친 후회와 귀환 강박에서 발생한다.
- 저승역 안내방송의 목적지 공백을 듣는 사람이 스스로 채운다.
- 안내가 끝나기 전에 이동하면 시간은 흐르지만 공간 위치가 승강장 시작점으로 초기화된다.
- 안정화는 공백 구간을 확인하고 공식 운행 기록과 식별음으로 안내 절차를 복원하는 방식이다.
- 회수 잔향은 `저승역 반복 안내 잔향`이다.

## 핵심 플레이 경험

- 메인 조사는 `상황 설명 → 선택 → 짧은 결과 → 기록 → 다음 질문`이 중심이다.
- 플레이어는 주인공 요원 한 명과 최대 두 명의 서포트를 편성하고, 현장 관찰을 읽어 최종 판단한다.
- 괴이는 처치하지 않고 단서로 규칙을 이해해 현재 출현을 안정화한다.
- 실패는 피해와 함께 다음 판단에 쓸 위험 사례를 남긴다.
- 사건 완료 뒤 플레이어가 알아낸 전조·금기·대응이 괴이 매뉴얼에 등록된다.
- 핵심 보상은 전리품보다 예상 적중, 정보 연결, 현장의 즉시 반응, 매뉴얼 완성이다.
- 일상 에피소드는 HQ 준비 화면의 선택형 요원 대화이며 필수 단서·안정화·엔딩 진행을 대체하지 않는다.
- 목표 플레이타임은 필수 125~155분, 선택 콘텐츠 포함 180~220분이며 아직 실측 전이다.

## 첫 10분 목표

- 시작 후 3분 안에 의미 있는 현장 선택을 제공한다.
- 10분 안에 단서 두 개를 연결하고 가설을 한 번 검증한다.
- 플레이어가 괴이가 왜 완전히 죽지 않는지 이해한다.
- 플레이어가 괴이 기록국이 왜 매뉴얼을 만드는지 이해한다.
- 실패해도 재시작보다 위험 사례와 다음 비교 근거가 남는다.
- 세력·연구·시장·모든 능력 수치·10일 종료 규칙은 첫 10분에 설명하지 않는다.

## 표현 원칙

- UI는 대시보드보다 배경·캐릭터·현재 행동이 먼저 보이는 장면 중심 구조를 사용한다.
- `기록관 아카 · 괴이 기록국 관제 AI`는 전문성 70·귀여움 30의 디지털 유령 마스코트다. 내부 `로그` ID와 저장 키만 호환용으로 유지한다.
- 대사는 일상에서 풍부하게, 조사에서는 짧게 사용한다.
- 피해·사망·후유증은 농담 소재로 삼지 않는다.
- 아카와 요원은 미확보 단서·정답·숨은 수치를 누설하지 않는다.
- 공식 용어 이관 전 내부 키·파일명은 호환을 위해 유지할 수 있으나 플레이어 노출 문구는 승인 용어를 기준으로 한다.

## 상태와 호환성

- `HIGH_RISK_DOMAIN`: 저장, 경제, 진행, 엔딩, 에피소드 의미 변경은 의미론적 고위험이다. 이는 파일 경로 자체를 보호하는 보호 경로와 구분한다.
- 일상은 완료 기록과 진행 중 카드 상태를 저장하며, `mvp-038` 저장에는 빈값을 보완한다. 그 외 UI 표현은 가능한 기존 상태에서 계산한다.
- 자동행동과 장비는 플레이어 선택이나 핵심 추리를 대체하지 않는다.
- 승인 설계의 튜토리얼 완료 여부는 기존 사건 진행 상태에서 계산하는 방식을 우선한다.
- 모든 저장을 불러올 때 권나래를 `selected_agent_ids` 첫 위치로 승격하고, 나머지 최대 두 ID를 서포트로 해석한다. 별도 역할 저장 필드는 추가하지 않는다.
- 반일 일정은 권나래 한 명만 필요·소비한다. 이관 시 현재 반일의 기존 주인공 일정을 권나래에게 복사하고 원본 일정은 보존한다.
- 저승역 첫 작전은 권나래 주인공, 오현·강이준 서포트로 고정한다.
- 회수는 전조를 자동 관측하고, 행동 직후 직전 결과와 다음 전조를 동시에 표시한다.
- 자동 지원 시도는 성공·실패 모두 기존 `used_agent_supports`에 기록해 재진입 재추첨을 막는다.
