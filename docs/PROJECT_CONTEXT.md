## 한 줄 정의

`괴이 기록국`은 인간의 마음과 기억에서 되살아나는 괴이의 규칙을 조사하고, 현재 출현을 안정화한 뒤 다음 피해를 막을 괴이 매뉴얼을 남기는 PC용 현대 오컬트 수사 어드벤처다.

- MVP-043 + CORE-VALIDATION-001 / Ver 4.2 / 저장 스키마 `mvp-039` (`mvp-038` 이관 지원)
- Godot 4.7 stable / GDScript / PC·Steam 16:9
- 최대 10일, 오전·오후 순차 반일 캠페인
- 구현 사건: 저승역, 비 오는 골목의 빨간 우산, 폐주파수 방송국
- 권나래 고정 주인공
- 초기 요원: 권나래·윤서하·오현·강이준·한유리
- 서포트: 주인공 외 최대 2명
- 외부 접점: 박도윤·이세린·레이먼드 케인·카밀라 바르가스
- 세력: 소문시장, 마도회, 퇴마사 계보
- 요원 능력: 제압·분석·방호·치료·교감
- HQ 일정 비소모 일상 에피소드 3장

상세 구현 목록과 미구현 계획의 구분은 `docs/CURRENT_STATUS.md`를 따른다.

## 공식 용어와 세계관

- 공식 기관명: **괴이 기록국**
- 사건 완료 표기: **안정화 상태**
- 실패 기록: **위험 사례**
- 회수 대상: **잔향**
- 최종 기록 보상: **괴이 매뉴얼 작성·갱신**
- 기록 상태: **공식 규칙** / **검증 대기 후보** / **위험 사례**
- 안내자 표시명: **기록관 아카 · 괴이 기록국 관제 AI**

괴이는 인간의 두려움·후회·집착·죄책감·기억·소문이 반복되어 현실에 비정상적 규칙으로 나타난 현상이다. 완전히 죽지 않으므로 기록국은 영구 제거가 아니라 현재 출현의 안정화와 재출현 대응 절차 기록을 목표로 한다.

## 핵심 플레이 경험

```text
상황 설명 → 현재 판단 질문 → 행동 선택 → 짧은 결과
→ 새 비교 근거·위험 사례 기록 → 다음 선택 → 안정화·잔향 회수 → 괴이 매뉴얼 갱신
```

- 플레이어는 상황과 단서를 읽고 규칙을 추론한다.
- 실패는 재시작 명령이 아니라 다음 판단에 쓸 위험 사례를 남긴다.
- 요원은 정답 버튼이나 수치 카드가 아니라 짧은 반응과 능력 발동으로 기여한다.
- 기록관 아카는 상황을 정리하지만 미확보 단서·정답·숨은 수치를 누설하지 않는다.
- 핵심 보상은 전리품보다 예상 적중, 정보 연결, 현장의 즉시 반응, 매뉴얼 완성이다.
- HQ 일상과 관계 이벤트는 필수 단서·안정화·엔딩 진행을 대체하지 않는다.

## 저승역 정체성

- 발생 감정: 막차를 놓친 후회와 귀환 강박
- 핵심 규칙: 안내방송의 목적지 공백을 듣는 사람이 자신의 귀환 욕망으로 채운다.
- 위험 규칙: 안내 종료 전에 이동하면 시간은 흐르지만 공간 위치가 승강장 시작점으로 초기화된다.
- 안정화: 공백 구간을 확인하고 공식 운행 기록과 식별음으로 안내 절차를 복원한다.
- 회수 잔향: `저승역 반복 안내 잔향`

v0.9 승인 설계는 MVP-043 구현과 GDD에 통합됐다. 당시 원문은 `docs/archive/backup/2026-07-16/CONTENT_DIRECTION_V09_BACKUP.md`에서 찾는다.

## 표현 원칙

- UI는 대시보드보다 배경·캐릭터·현재 행동이 먼저 보이는 장면 중심 구조를 사용한다.
- 대사는 일상에서 풍부하게, 조사에서는 한 문장·최대 두 줄 중심으로 사용한다.
- 피해·사망·후유증은 농담 소재로 삼지 않는다.
- 아카와 요원은 미확보 단서·정답·숨은 수치를 누설하지 않는다.
- 내부 `로그` ID와 저장 키는 호환용으로 유지할 수 있지만 플레이어 노출은 `기록관 아카`를 사용한다.
- 관계는 숫자형 연애 호감도가 아니라 신뢰·경계·부채·경쟁·공감·보호·균열 같은 선택 기억으로 표현한다.

## 상태와 호환성

- `HIGH_RISK_DOMAIN`: 저장, 경제, 진행, 엔딩, 에피소드 의미 변경
- 모든 저장에서 권나래를 `selected_agent_ids[0]`로 승격하고 나머지 최대 두 ID를 서포트로 해석한다.
- 반일 일정은 권나래 한 명만 필요·소비한다.
- 기존 저장 이관 시 현재 반일의 과거 주인공 일정을 권나래에게 복사하고 원본 일정·진행 중 작전을 보존한다.
- 저승역 첫 작전은 권나래 주인공, 오현·강이준 서포트로 고정한다.
- 자동 지원 시도는 성공·실패 모두 기존 상태에 기록해 재진입 재추첨을 막는다.
- UI·표정·컷인 연출은 저장·진행 상태의 소유자가 되지 않는다.
