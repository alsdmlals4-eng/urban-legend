## 1. 프로젝트 한 줄 약속

> 괴이의 규칙을 조사해 전조를 읽고, 턴제 회수 현장에서 추론을 증명하며, 성공과 실패를 다음 출현의 생존 매뉴얼로 직접 기록하는 현대 오컬트 수사 RPG.

## 2. 시장 포지션

가장 가까운 직접 비교 대상은 기관형 도시괴담 조사 어드벤처다. 따라서 `도시괴담을 조사하는 기관`만으로는 차별화가 부족하다.

> **괴담의 진실을 밝히는 게임이 아니라, 다시 나타날 괴이에서 살아남을 규칙을 직접 기록하고 전투에서 증명하는 팀 수사 게임.**

차별화의 중심은 다음 세 가지다.

1. 플레이어가 규칙 가설과 근거 관계를 직접 만든다.
2. 조사 결과가 회수 전투의 전조 정보 우위로 즉시 변환된다.
3. 성공과 실패가 모두 괴이 매뉴얼의 다음 판단 자산이 된다.

## 3. 대상 플레이어

- 오컬트·도시괴담과 미스터리 서사를 좋아하는 플레이어
- 관측한 정보를 비교하고 이유를 설명하는 추리를 선호하는 플레이어
- 실시간 액션보다 낮은 입력 부담의 턴제 판단을 선호하는 플레이어
- 짧은 사건 안에서 조사-회수-기록의 완결을 원하는 PC/Steam 플레이어

초기 PoC는 대규모 경영·파밍·하드코어 전투 플레이어를 주 대상으로 하지 않는다.

## 4. 핵심 감정 흐름

```text
의심
→ 관찰과 비교
→ 가능성 배제
→ 규칙 가설 작성
→ 합리적 확신
→ 약간의 위험을 감수한 검증
→ 괴이 반응과 이해 갱신
→ 회수 전투의 전조 해석
→ 준비한 대응의 적중
→ 포획 창
→ 괴이 매뉴얼 승격
```

## 5. 뾰족한 재미

> **조사에서 읽은 규칙이 회수 전투의 전조로 실제 나타나고, 준비한 대응으로 포획 창을 여는 순간.**

제1 재미는 규칙을 간파하는 추론 쾌감이다. 불완전한 정보에서의 검증이 긴장을 만들고, 괴이 매뉴얼과 더 나은 서사 결과가 장기 보상을 만든다.

## 6. 최소 코어의 여섯 기둥

### 6.1 관측 가능한 페어플레이 정보

핵심 단서와 선택지의 모순은 플레이어가 확인 가능한 정보로 판정한다. 요원·장비·기록관 아카·자동행동은 정답을 대신하지 않는다.

### 6.2 플레이어 작성형 규칙 가설

플레이어는 남은 가설에 지지 근거, 반박 근거, 미해결 질문을 연결한다. 괴이 매뉴얼은 결과 요약이 아니라 실제 추론 작업 도구다.

### 6.3 합리적 확신 뒤 위험 검증

조사를 많이 해도 위험이 완전히 사라지지는 않는다. 플레이어는 충분한 근거를 만든 뒤 현장 검증으로 남은 불확실성을 갱신한다.

### 6.4 조사 지식의 전투 정보 우위

현장 이해도는 공격력 보너스가 아니라 회수 전투의 행동명·대상·범위·발동 조건을 읽는 정보 우위다.

### 6.5 패턴 대응형 회수

괴이는 HP를 모두 깎아 처치하지 않는다. 전조를 읽고 적절한 대응을 누적해 안정화·봉쇄·포획 조건을 연다.

### 6.6 성공과 실패의 영구 기록

성공은 공식 규칙과 대응 절차로, 실패는 위험 사례와 배제 근거로 남는다. 둘 다 다음 출현의 실제 판단 자산이다.

## 7. 페어플레이 방향

### 논리로 결정

- 단서 내용과 선택지 모순
- 규칙 가설의 지지·반박 관계
- 현장 이해도 승격
- 괴이의 실제 패턴과 포획 조건

### 확률로 결정 가능

- `단서 확보`·`유력` 단계에서 이미 관측한 패턴의 세부 전조를 이번 턴에 읽는가

확률 판정은 거짓 정보를 만들지 않는다. 실패 시 `놈은 무언가 하려 한다.`만 표시하고, `이해` 단계에서는 관측 패턴을 확정 해석한다.

## 8. 조사 페이즈 방향

```text
상황 제시
→ 대응 선택지 4개
→ 관련 매뉴얼 사례 2~3개 자동 표시
→ 기록을 선택지에 연결
→ 모순되는 2개 배제
→ 남은 경쟁 가설에 근거 관계 작성
→ 현장 검증
→ 현장 이해도·위험 사례 갱신
```

- 자료 검색은 자동화하지만 의미 판정은 플레이어가 한다.
- 배제 시도에는 체력·위험도 비용이 없다.
- 실제 현장 대응에서만 비용이 발생한다.
- 오답 뒤 선택지는 `기존 가설 1개 + 직전 반응에서 생긴 신규 가설 1개`로 부분 갱신한다.
- 위험도 최대에서는 추가 검증을 차단하고 긴급 회수로 전환한다.

## 9. 지식의 두 층

### 괴이 매뉴얼 지식 - 영구

공식 규칙, 관측 패턴, 위험 사례, 연구 확정 정보는 괴이별로 유지한다.

### 현장 이해도 - 현재 출현별

`미확인 → 단서 확보 → 유력 → 이해`

현재 장소·환경·변형·발현 조건을 반영한다. 같은 괴이가 다시 나타나도 완전한 초면은 아니지만 현재 출현은 새로 조사한다. 다른 괴이와의 유사성으로 자동 전이하지 않는다.

## 10. 회수 페이즈 방향

- 턴제 패턴 대응 전투
- 실패 문구: `놈은 무언가 하려 한다.`
- 성공 문구: `놈은 [행동명]을 하려 한다.`
- 행동 문법: 관찰 / 방어·보호 / 지원 / 규칙 대응 / 봉쇄·포획
- 승리 조건: HP 0이 아니라 포획 창 개방
- 체력 역할: 실수와 검증을 감당하는 실패 예산

### 미관측 핵심 패턴

- 첫 발동은 행동명을 공개하지 않는다.
- 범용 방어·엄폐·보호 중 최소 하나가 항상 유효하다.
- 즉사, 강제 중상, 캠페인 소프트락, 히든 분기 영구 실패를 만들지 않는다.
- 첫 발동 뒤 매뉴얼에 등록하고 두 번째부터 이해도 판정을 적용한다.

## 11. 주인공과 요원 방향

### 권나래

- 고정 주인공이며 교체 기능이 없다.
- 권나래의 일정만 진행·소비한다.
- 판단과 기록의 책임을 맡는다.
- 부상 상태에서도 강행 또는 회복을 선택할 수 있다.

### 서포트

- 최대 2명, 독립 일정 없음
- 정답·필수 단서 독점 금지
- 관찰 질문·위험 해석·결과 반응으로 차별화
- 수치는 피해·지원 강도와 복구만 조절
- 자동 최적 요원 선택 금지

### 기록관 아카

- 관련 매뉴얼 사례와 기록 위치를 안내
- 미확보 단서·정답·숨은 수치 누설 금지
- 시스템은 자료를 찾고 플레이어는 의미를 판정

## 12. 승인된 지원 방향

다음 요소는 코어를 증폭하지만 PoC 뒤 세부 범위를 조정할 수 있다.

- 체력·부상·회복과 3축 결과 등급
- 포획·연구·괴이 전용 대응
- 기간제 챕터와 핵심 사건 마감
- 연계 의뢰 최대 1개 + 독립 의뢰 1개
- 신뢰도·잔향 파편·연구도
- 히든 분기와 가치관 엔딩
- 일상·후일담·관계·표정·컷인
- 기존 미니게임의 조건부 재활용

지원 시스템은 첫 사건의 조사-가설-전조-포획 인과를 흐리면 축소하거나 보류한다.

## 13. 지원 시스템 가드레일

### 부상·결과

- 권나래는 중상에서도 강행 또는 반일 회복 가능
- 약한 자동 회복 누적
- 서포트 중상은 일시 동행 제한 가능
- 소프트락 금지
- 결과 등급은 회수 품질, 피해 관리, 지식 품질의 3축

### 포획·연구·히든 분기

- 포획 중에는 해당 괴이가 배치되지 않음
- 연구는 정보 확정과 제한적 전용 대응 제공
- 히든 분기는 새로운 장면·기록·더 희망적인 결과를 열 수 있음
- 히든 분기는 기관 평가·자원·포획 안정성까지 항상 우월한 상위 정답이 아님

### 챕터·의뢰

- 챕터당 핵심 사건 1개와 마감
- 연계 의뢰 최대 1개 + 독립 의뢰 1개
- 의뢰는 정답·필수 단서·이해도를 직접 지급하지 않음
- 미니게임은 `전조 관측 → 규칙 적용 → 즉시 결과` 공통 인지 문법을 충족할 때만 재활용

## 14. 벤치마킹 결론

- **Urban Myth Dissolution Center:** 기관형 도시괴담과 정황 증거는 참고하되, 장기 매뉴얼·팀·실패 학습·회수 전투 인과로 차별화
- **PARANORMASIGHT:** 구체적 지역성과 인물 갈등을 한국 도시 장소·공공 기록과 결합
- **Golden Idol:** 관찰 정보를 플레이어가 직접 이론으로 조립하는 원리를 규칙 가설 카드로 채택
- **Return of the Obra Dinn:** 기록장을 실제 추론 도구로 사용하는 원리를 매뉴얼 불확실성·반증 기록에 적용
- **No Case Should Remain Unsolved:** 첫 공개 빌드 범위를 줄이고 감정적 종결 강화
- **Lobotomy Corporation:** 개체별 규칙과 정보의 힘을 적용하되 영구 사망·반복 노동·시설 경영 제외
- **Disco Elysium:** 실패가 다른 정보·인물 반응을 남기는 원리 적용, 핵심 추리 난수화 제외
- **WORLD OF HORROR:** 모듈형 괴담과 강한 시각 언어를 참고하되 로그라이트 랜덤·전투 중심 제외

상세 근거와 ADOPT / ADAPT / AVOID 판정은 `PROJECT_CORE_STRESS_TEST_AND_BENCHMARK.md`를 따른다.

## 15. MVP 재기준화

### 현재 구현선

MVP-043 + CORE-VALIDATION-001 + UX-PD-001 2A / Ver 4.2 / save `mvp-039`는 실제 구현 기준선으로 유지한다.

## 16. 성공 판단

- 플레이어가 왜 두 선택지를 배제했는지 근거로 설명하는가?
- 가설의 지지·반박 관계를 설명하는가?
- 조사 결과가 회수 전투의 정보 우위로 체감되는가?
- 예측 성공 뒤에도 대응 선택이 남는가?
- 오답 뒤 새 근거로 판단을 갱신하는가?
- 미관측 핵심 패턴이 위험하지만 대응 가능하다고 느끼는가?
- 매뉴얼을 시스템 정답집이 아니라 자신이 만든 기록으로 인식하는가?

## 17. Production gate

문서 승인과 시장·적대적 검토는 끝났지만 신규 플레이어 행동 증거는 없다. CORE-MVP-001의 사전 선언 지표를 통과하기 전에는 캠페인·의뢰·연구·서사 확장을 시작하지 않는다.
