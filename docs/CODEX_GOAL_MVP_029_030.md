# MVP-029~030 Codex Goal

## 목표

두 기존 사건을 PC 16:9에서 처음부터 끝까지 플레이할 때 대화, 텍스트 노벨형 조사, 짧은 현장 판정, 안정화/회수, 보고서, DB와 저장이 한 흐름으로 이어지게 한다.

## 구현 범위

- 기본 viewport를 PC 16:9에 맞춘다.
- 저승역에는 축소 원 타이밍 판정, 빨간 우산에는 방향키 비 피하기 판정을 구현한다.
- 성공/실패를 조사 상태, 회수 조건, 보고서와 DB에 기록한다.
- 현재 플레이 용어를 현장 판정과 안정화/회수 중심으로 맞춘다.
- README에 Steam 데모 후보 범위와 소개문을 정리한다.

## 제외 범위

- 새 사건, 복잡한 리듬/액션 게임, 대형 아트, 파티 전투, 저장 마이그레이션, 모바일 최적화, HTML 대시보드

## 완료 확인

- 두 사건의 판정은 짧고 실패해도 진행이 막히지 않는다.
- 판정 결과가 회수, 사건 보고서, DB, 저장/이어하기에 남는다.
- `Ver 3.0`, PC 16:9, 요원 팀 중심, 안정화/회수 용어가 일치한다.
- 전체 프로젝트와 핵심 씬 headless 검증 및 수동 플레이 미확인 항목을 보고한다.
