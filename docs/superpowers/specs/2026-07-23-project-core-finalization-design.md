# 괴이 기록국 프로젝트 코어 확정 설계

> 승인일: 2026-07-23  
> 승인 상태: `CORE_RECORDED`  
> 검토 상태: `CORE_STRESS_TESTED`  
> 구현 상태: `POC_PENDING`  
> Production gate: `HOLD_UNTIL_PLAYER_EVIDENCE`

## 1. 확정 결론

> 관측 가능한 단서로 괴이의 규칙 가설을 만들고, 위험을 감수한 검증으로 이해를 갱신한 뒤, 그 이해로 회수 전투의 전조를 읽어 포획 조건을 열고, 성공과 실패를 다음 출현의 괴이 매뉴얼로 기록하는 현대 오컬트 수사 RPG.

뾰족한 재미:

> 조사에서 읽은 규칙이 회수 전투의 전조로 실제 나타나고, 준비한 대응으로 포획 창을 여는 순간.

## 2. 최소 코어

1. 관측 가능한 단서와 매뉴얼을 비교한다.
2. 플레이어가 규칙 가설과 지지·반박·미해결 근거를 직접 구성한다.
3. 불완전한 가설에 위험을 걸어 검증하고 현장 이해도를 갱신한다.
4. 현장 이해도가 회수 전투의 전조 정보 우위로 바뀐다.
5. 턴제 대응으로 포획 조건을 열며 괴이를 처치하지 않는다.
6. 성공과 실패를 공식 규칙·후보·위험 사례로 매뉴얼에 남긴다.

## 3. 조사 계약

- 텍스트 노벨형 상황과 4개 대응 선택지
- 관련 괴이 매뉴얼 사례 2~3개 자동 제시
- 플레이어가 기록을 선택지에 연결
- 논리적 모순으로 2개 배제
- 남은 2개는 모두 합리적인 경쟁 가설
- 규칙 가설 카드: 규칙 문장, 지지, 반박, 미해결, 검증 결과, 기록 상태
- 오답은 정답을 공개하지 않고 반응 단서·배제 근거·현상 심화를 남김
- 오답 뒤 `기존 가설 1개 + 신규 변형 가설 1개`로 부분 갱신
- 반복 검증은 체력·위험도·현상 심화로 제한
- 위험도 최대에서 긴급 회수로 전환

## 4. 페어플레이와 확률

### 결정론적 영역

- 단서 내용과 획득 계약
- 기록과 선택지의 모순
- 가설의 지지·반박 관계
- 현장 이해도 승격
- 실제 패턴과 포획 조건

### 확률 허용 영역

- `단서 확보`·`유력` 단계에서 이미 관측한 패턴의 행동명을 이번 턴에 읽는가
- 실패 문구는 `놈은 무언가 하려 한다.`
- 거짓 행동명·대상·범위 금지
- `이해` 단계는 관측 패턴 확정 해석
- 저장·불러오기로 재추첨 금지

## 5. 지식 모델

### 영구 괴이 매뉴얼

- 공식 규칙
- 관측 패턴
- 위험 사례
- 연구 확정 정보와 전용 대응

### 현재 출현의 현장 이해도

`미확인 → 단서 확보 → 유력 → 이해`

재출현 때 영구 지식은 유지하지만 현장 이해도는 현재 장소·환경·변형 조건에 맞게 다시 쌓는다. 다른 괴이로 자동 전이하지 않는다.

## 6. 회수 전투 계약

- 전조 기반 턴제 패턴 대응 전투
- 행동: 관찰 / 방어·보호 / 지원 / 규칙 대응 / 봉쇄·포획
- 조사 보상은 공격력 증가가 아니라 행동명·대상·범위·발동 조건의 정보 우위
- 체력은 실패 예산
- 승리는 HP 0이 아니라 대응 누적으로 포획 창을 여는 것

### 미관측 핵심 패턴

- 첫 발동 행동명 비공개
- 미관측 핵심 전조라는 사실은 표시
- 범용 방어·엄폐·보호 중 최소 하나는 항상 유효
- 즉사·강제 중상·소프트락·영구 히든 실패 금지
- 첫 발동 뒤 매뉴얼 등록, 두 번째부터 이해도 해석

## 7. CORE_SUPPORT

다음은 승인된 방향이지만 최소 코어가 아니며 PoC 뒤 세부 범위를 조정한다.

- 권나래 고정 주인공, 단일 일정 주체, 최대 2인 서포트
- 체력·부상·회복과 3축 결과 등급
- 포획·연구·괴이 전용 대응
- 기간제 챕터와 핵심 사건 마감
- 연계 의뢰 1개 + 독립 의뢰 1개
- 신뢰도·잔향 파편·연구도 소량 보상
- 조사에서 조건을 만들고 회수에서 실행하는 히든 분기
- 가치관 축과 핵심 사건 보정 엔딩
- 기록관 아카, DB, 보고서, 일상·후일담·컷인

## 8. 지원 시스템 가드레일

- 권나래는 교체하지 않으며 중상에서도 강행/회복 선택 가능
- 약한 자동 회복, 진행 소프트락 금지
- 결과 등급은 회수 품질·피해 관리·지식 품질
- 히든 분기는 더 희망적일 수 있으나 항상 높은 기관 평가·자원·등급을 주는 상위 정답이 아님
- 의뢰는 핵심 사건 정답·필수 단서·이해도를 직접 지급하지 않음
- 기존 미니게임은 `전조 관측 → 규칙 적용 → 즉시 결과` 문법을 충족할 때만 재사용
