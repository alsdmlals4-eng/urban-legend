# Reviews

이 폴더는 MVP 검수, PR 검토, Codex 결과 리뷰를 보관하는 위치다.

GitHub PR 설명이 1차 기록이고, 별도 정리가 필요한 큰 작업만 이 폴더에 리뷰 문서를 남긴다.

## 파일명 규칙

```text
MVP_019_review.md
PR_000_review.md
bug_YYYYMMDD_short-title_review.md
```

## 표준 구조

- PR:
- MVP:

-

-

- [ ] 모바일 세로 UI 확인

## 작성 원칙

- 구현 완료를 추정하지 않는다.
- 실제 PR diff, 실행 결과, 파일 내용을 근거로 판단한다.
- 검증하지 못한 항목은 실패가 아니라 `미검증`으로 분리한다.
- 기획 의도와 맞지 않는 정상 동작은 승인하지 않는다.
