## 재현 경로

신규 10일 캠페인에서 저승역을 Day 1 오전에 시작해 HQ로 중단·저장·재개했다. Day 2 오전에 빨간 우산을 완료하고 세력 의뢰 1건을 오현으로 처리했다. 이후 위험도 주입·테스트 훅 없이 휴식 일정을 진행해 폐주파수 방송국이 Day 8 오전에 `lead`로 발견되는 것을 확인했다. 해당 사건에서는 실제 조사 장면의 결과 블록, 첫 회수 패턴의 오대응, 오대응 학습, 올바른 3패턴 회수, 결과 보고서·DB·저장 왕복을 실행했다.

```powershell
& "C:\Users\user\Downloads\Godot_v4.7-stable_win64.exe\Godot_v4.7-stable_win64_console.exe" --headless --path . --script res://tests/test_three_case_campaign_manual_qa.gd
```

결과: **135 passed, 0 failed**. 테스트 종료 시 Runtime UI Editor의 장면 해제, RID·ObjectDB 누수 경고가 있었으나 실행 실패·저장 훼손은 없었다. 기존 경고와 같은 종류이므로 별도 후속 후보로 유지한다.
