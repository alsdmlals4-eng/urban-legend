# 텍스트 노벨·선택 서사 벤치마킹과 MVP-039

## 범위

- 확인일: 2026-07-14
- 목적: 텍스트 선택형 게임의 선택 맥락, 실패 피드백, 기록 동기를 `괴담기록국`의 두 구현 사건 UX에 반영한다.
- 반영 범위: 조사·회수·결과 화면의 기존 데이터 재표현과 문서 갱신만 포함한다.
- 제외 범위: 저장 스키마, `GameState`, 사건 JSON, 캠페인 일정, 새 분기·단서·재화의 변경.
- 평점·리뷰 수는 확인 시점의 스토어 표본이다. 순위 자체를 일반화하지 않고 설계 관찰의 보조 근거로만 쓴다.

두 구현 사건의 성공·오대응 경로를 1280×720과 1920×1080에서 수동 검증한다. 이해가 되지 않는 항목만 다시 조정하며, 신규 단서·분기·저장값 추가는 MVP-040 설계 승인 전까지 보류한다.
