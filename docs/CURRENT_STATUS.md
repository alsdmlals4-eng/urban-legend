## 구현되지 않은 항목

다음은 승인 목표이며 현재 구현 완료가 아니다.

- 턴제 패턴 대응형 회수 전투
- 이해도 단계와 전조 해석 확률
- 조사 4선택지·매뉴얼 근거 연결·자동 관련 사례 2~3개
- 위험도 한계의 긴급 회수 진입
- 권나래 중상 강행·회복 선택과 자동 회복
- 포획 중 배치 제외와 연구 단계
- 연구 기반 조사·회수 히든 분기
- 기간제 챕터, 조기 해결 정리 기간, 마감 긴급 회수
- 가치관 기반 캠페인 엔딩

## 현재 권장 구현 순서

1. 문서 정합성과 기존 세 사건·저장·ID 보호 범위를 감사한다.
2. `CORE-MVP-001`을 독립 실험 경로로 구현한다.
3. 신규 플레이어 5~8명에게 조사 근거 배제와 전조 예측 연결을 관찰한다.
4. PoC 통과 뒤 `CORE-MVP-002`의 부상·포획·연구를 구현한다.
5. `CORE-MVP-003`에서 기간제 챕터와 기존 미니게임 의뢰를 연결한다.
6. `CORE-MVP-004`에서 연구 히든 분기와 엔딩 슬라이스를 검증한다.
7. 저장 스키마·기존 사건 이관·MVP-044~046 재매핑을 별도 승인한다.

## 불변 조건

- 권나래는 고정 주인공이며 교체하지 않는다.
- 권나래의 일정만 소비하고 서포트는 독립 일정을 요구하지 않는다.
- 조사와 회수 전투는 이해도를 통해 직접 연결된다.
- 시스템은 관련 자료를 찾지만 정답과 대응을 대신 선택하지 않는다.
- 실패는 체력·부상·위험도 비용과 함께 새 정보와 위험 사례를 남긴다.
- 괴이는 전통 RPG의 처치 대상이 아니라 회수 조건을 충족할 대상이다.
- 이해도는 공격력 보너스가 아니라 정보 우위다.
- 다른 괴이와 유사성으로 지식을 자동 전이하지 않는다.
- 의뢰와 연구가 핵심 정답을 판매하지 않는다.
- 구현되지 않은 목표를 구현 완료로 기록하지 않는다.
