## 1. 게임 약속

| 질문 | 답 |
|---|---|
| 플레이어는 누구인가 | 괴이 기록국의 현장 판단 담당 요원 권나래 |
| 조사에서 무엇을 하는가 | 단서와 괴이 매뉴얼로 4개 선택지 중 2개를 배제하고 경쟁 가설을 검증한다 |
| 전투에서 무엇을 하는가 | 이해도에 따라 전조를 읽고 턴제 대응으로 회수 조건을 충족한다 |
| 무엇을 남기는가 | 확정 규칙·관측 패턴·위험 사례·연구 결과가 축적되는 괴이 매뉴얼 |
| 장기 목표는 무엇인가 | 더 깊은 이해로 새로운 해결 방식과 여러 캠페인 결말을 연다 |
| 무엇을 하지 않는가 | 자동 정답, 거짓 예측, 조사와 무관한 화력전, 권나래 교체 |

## 2. 핵심 루프

```text
챕터 일정 계획
→ 핵심 사건 조사
→ 매뉴얼 근거로 선택지 배제
→ 가설 검증과 이해도 축적
→ 전조 기반 턴제 회수 전투
→ 결과 등급·포획
→ 연구·매뉴얼 갱신
→ 의뢰·회복·후일담
→ 다음 챕터 또는 재출현 히든 분기
```

| 항목 | 계약 |
|---|---|

| 시간 단위 | 오전·오후 순차 반일 |
| 주인공 일정 | 권나래의 일정만 필요·소비 |
| 서포트 | 최대 2명, 독립 일정 불필요 |

| 조기 해결 | 신규 대형 사건 없이 회복·연구·후일담 정리 기간 |

## 4. 조사 페이즈

### 4.1 화면 구조

1. 상황·전조 설명
2. 대응 선택지 4개
3. 자동 제시된 관련 괴이 매뉴얼 사례 2~3개
4. 현재 체력, 괴이 위험도, 현상 심화 제약

### 4.2 선택지 구성

| 구분 | 수량 | 판별 방식 |
|---|---:|---|
| 배제 가능한 오답 | 2 | 확정 규칙·위험 사례·현재 근거와의 모순으로 배제 |
| 경쟁 가설 | 2 | 현재 정보만으로 확정 불가, 현장 검증 필요 |

배제 가능한 오답도 장르 상식으로 쉽게 고르는 농담 선택지가 아니라, 일부 기록에는 맞지만 현재 전조 하나와 충돌하는 합리적 오답이어야 한다.

### 4.3 근거 연결

- 플레이어가 매뉴얼 기록을 선택지에 직접 연결한다.
- 근거가 타당하면 해당 선택지를 지울 수 있다.
- 부적합한 연결은 `이 기록만으로는 배제할 수 없다`고 표시한다.
- 배제 시도 자체에는 체력·위험도 비용이 없다.
- 시스템은 관련 자료를 찾아주지만 정답을 추천하지 않는다.

### 4.4 가설 검증

```text
경쟁 가설 2개
→ 1개 실행
→ 괴이 반응
→ 이해도·체력·위험도 갱신
→ 성공 또는 위험 사례 기록
```

오답이면 정답을 공개하지 않는다. 반응 단서, 배제 근거, 현상 심화 상태만 제공한다.

### 4.5 실패 뒤 선택지 갱신

오답 가설은 제거한다. 남은 기존 가설 1개와 현상 심화로 생긴 신규 변형 가설 1개를 다시 비교한다. 신규 가설은 직전 반응으로 설명되어야 하며 무작위로 등장하지 않는다.

### 4.6 괴이 위험도

검증 반복은 가능하지만 다음 비용이 누적된다.

- 권나래·서포트 체력 감소
- 부상 가능성 증가
- 접근 지점 차단
- 지원 행동 제한
- 민간 피해 위험 증가
- 기존 대응 조건 변화
- 사건 결과 등급 상한 하락 가능

위험도 최대에서는 추가 검증을 차단하고 현재 이해도로 긴급 회수 전투에 진입한다.

## 5. 이해도

### 5.1 단계

| 단계 | 조사 의미 | 회수 전투 효과 |
|---|---|---|
| 미확인 | 유효한 단서 부족 | 전조 해석 불가 |
| 단서 확보 | 일부 조건 확인 | 낮은 확률로 관측 패턴 해석 |
| 유력 | 경쟁 가설이 크게 좁혀짐 | 높은 확률로 관측 패턴 해석 |
| 이해 | 규칙과 반증을 충분히 설명 | 관측 패턴 확정 해석 |

### 5.2 영속 범위

- 괴이별 확정 규칙, 관측 패턴, 위험 사례, 연구 결과는 영구 보존한다.
- 현재 출현의 환경, 변형, 발현 조건은 재출현 때 다시 조사한다.
- 다른 괴이와의 유사성으로 이해도를 자동 전이하지 않는다.

## 6. 회수 페이즈

### 6.1 전투 형식

턴제 패턴 대응 전투다. 핵심 질문은 `다음 전조가 무엇을 의미하며 어떤 대응을 해야 하는가`다.

### 6.2 전조 판정

| 결과 | 표시 |
|---|---|
| 실패 | `놈은 무언가 하려 한다.` |
| 성공 | `놈은 [행동명]을 하려 한다.` |

성공 시 조사로 확인한 범위 안에서 대상, 범위, 발동 조건을 함께 표시할 수 있다. 대응 행동은 추천하지 않는다.

### 6.3 핵심 패턴 최초 관측

- 첫 발동은 이해도와 관계없이 행동 확인 불가
- 미관측 핵심 전조임은 별도로 표시
- 방어·엄폐·보호 같은 범용 대응 가능
- 범용 대응은 전용 대응보다 효율이 낮음
- 실제 발동 뒤 패턴을 매뉴얼에 등록
- 두 번째 발동부터 이해도 판정 적용

### 6.4 회수 조건

괴이는 일반 적처럼 사망하지 않는다. 올바른 대응을 누적해 약화, 봉쇄, 안정화 조건을 만든 뒤 회수 행동을 실행한다.

## 7. 체력·부상·회복

### 권나래

| 상태 | 효과 |
|---|---|
| 정상 | 기본 상태 |
| 경상 | 전투 위험 관리와 결과 등급에 작은 불이익 |
| 중상 | 일정과 현장 강행 가능, 추가 부상·등급 하락 위험 크게 증가 |

- 반일 회복 일정으로 한 단계를 빠르게 완화한다.
- 다른 일정을 진행해도 약한 자동 회복이 누적된다.
- 권나래 부상으로 일정 자체를 차단하지 않는다.

### 서포트

- 경상: 지원 효과 감소
- 중상: 1~2회 일정 동안 현장 동행 불가
- 지원 요원의 부재는 다른 편성·장비·대응을 선택하게 하는 변수다.

## 8. 사건 결과 등급

세 축을 독립 계산한 뒤 종합한다.

| 축 | 평가 요소 |
|---|---|
| 회수 품질 | 회수 성공, 임시 봉합 여부, 잔류 위험 |
| 피해 관리 | 권나래·서포트 부상, 민간 피해, 무리한 강행 |
| 지식 품질 | 근거 확보, 가설 정확성, 공식·후보·위험 사례 구분 |

성공과 이해를 분리한다. 우연히 회수했더라도 잘못된 매뉴얼 기록이 있으면 낮은 등급을 받을 수 있다.

## 9. 포획과 연구

### 9.1 포획

- 회수 성공 괴이는 일정 기간 포획 상태가 된다.
- 포획 중에는 사건 배치 대상에서 제외한다.
- 포획 기간은 연구 기회를 제공하는 동시에 같은 괴이의 즉시 재등장을 막는다.

### 9.2 연구 보상

1. 매뉴얼 정보 확정
2. 연구도 축적
3. 중요 단계에서 괴이 전용 대응 해금

전용 대응은 일반 능력보다 강한 수치 보너스가 아니라 새로운 조건과 행동을 연다.

## 10. 히든 분기

```text
포획 연구
→ 전용 지식·대응 해금
→ 재출현 조사에서 히든 선택지
→ 특정 조건 성립
→ 회수 전투에서 전용 행동
→ 새로운 장면·기록·결과
```

- 조사에서 조건을 만들고 회수에서 실행한다.
- 실패하면 일반 회수 루트로 복귀할 수 있지만 비용은 남는다.
- 일반 회수보다 반드시 효율적이지 않다.
- 주요 보상은 새로운 장면, 기록, 괴이·일반인·세계관에 더 나은 결말 가능성이다.

## 11. 세력 의뢰

### 11.1 구성

| 유형 | 수량 | 역할 |
|---|---:|---|
| 핵심 사건 연계 의뢰 | 최대 1 | 이해도, 전조 관측, 피해자 구출 여부 등에 반응 |
| 독립 세력 의뢰 | 1 | 다른 지역·세력·세계관 확장 |

### 11.2 미니게임

- 기존 `route restore`, `rain dodge` 등을 우선 재활용한다.
- 조작 규칙은 유지하고 목표, 제약, 서사 맥락을 바꾼다.
- 한 의뢰에 핵심 조작 하나만 사용한다.
- 같은 챕터의 두 의뢰는 같은 미니게임을 반복하지 않는다.

### 11.3 보상

- 세력별 기본 신뢰도
- 소량의 잔향 파편 또는 연구도
- 우수 등급: 잔향 파편, 연구도, 추가 신뢰도 중 하나 선택
- 핵심 사건의 이해도나 정답은 직접 제공하지 않는다.
- 간접 지원: 치료, 매뉴얼 참고 범위, 전투 소모품, 연구 비용 완화, 피해자 보호 조건

## 12. 엔딩

### 가치관 축

- 괴이 구제
- 민간인 보호
- 기관 명령 준수
- 진실 공개

각 축은 선악 수치가 아니다. 히든 분기와 핵심 사건 결정이 큰 변화를 만들고 일반 사건·의뢰 선택이 작은 변화를 만든다. 최종 결말은 권나래, 기록국, 괴이, 일반인, 세계의 상태를 조합한다.

## 13. UI·접근성

| 화면 | 핵심 정보 |
|---|---|
| 조사 | 현재 상황, 4개 선택지, 매뉴얼 사례 2~3개, 체력·위험도 |
| 회수 | 괴이 전조, 예측 결과, 요원 상태, 대응 행동, 회수 조건 |
| 결과 | 회수 품질, 피해 관리, 지식 품질, 매뉴얼 변화 |
| 연구 | 포획 상태, 연구 단계, 확정 정보, 전용 대응 |

- 긴 한국어 자동 줄바꿈과 1280×720 가독성을 보장한다.
- 색상·음향만으로 판정과 위험을 전달하지 않는다.
- UI는 상태를 표현하지만 저장·진행 결과를 대신 소유하지 않는다.

## 14. 저장·호환 위험

승인 목표는 저장 스키마, 캠페인, 경제, 엔딩 의미를 변경하므로 고위험 구현이다. `CORE-MVP-001` PoC에서는 기존 세 사건과 저장을 직접 마이그레이션하지 않고 독립 플래그 또는 실험 경로로 검증한다. 실제 통합 전에는 새 저장 버전과 이전 저장 이관 정책을 별도 승인한다.

## 15. MVP 재기준화

### CORE-MVP-001 - 조사·이해도·회수 PoC

- 단일 사건 15~25분
- 4개 선택지, 배제 근거 2개, 경쟁 가설 2개
- 이해도 4단계
- 관측 패턴 2개와 미관측 핵심 패턴 1개
- 턴제 회수 5~8턴
- 성공·오답·긴급 회수 경로

### CORE-MVP-002 - 실패·부상·결과·포획·연구

- 권나래·서포트 부상
- 회복 일정과 약한 자동 회복
- 3축 결과 등급
- 포획 상태와 연구 2단계
- 전용 대응 1개

### CORE-MVP-003 - 챕터·의뢰

- 제한 반일 챕터 1개
- 핵심 사건 1개
- 연계 의뢰 1개, 독립 의뢰 1개
- 기존 미니게임 2종 재활용
- 신뢰도·잔향 파편·연구도 보상
- 조기 해결 정리 기간과 마감 긴급 회수

### CORE-MVP-004 - 히든 분기·엔딩 슬라이스

- 연구 전용 대응으로 조사 히든 선택지 해금
- 회수 전용 행동 실행
- 일반/히든 결과 비교
- 가치관 축 4개 중 최소 2개 반영
- 캠페인 결말 장면 2종 이상

## 16. 기존 계획 처리

MVP-044~046의 서사·관계·연출 자산과 문서는 폐기하지 않는다. 다만 새 조사·회수·챕터 구조가 확정되기 전 구현을 보류하고 다음처럼 재매핑한다.

- 사건 대사와 아카 안내: CORE-MVP-001 조사·회수 수직 슬라이스에 우선 적용
- 관계 태그: CORE-MVP-003 의뢰·후일담과 CORE-MVP-004 엔딩에 재검토
- 표정·컷인: 상태 비소유 원칙을 유지하며 조사·회수 핵심 판정 순간에 제한 적용

## 17. 검증 기준

- 플레이어가 근거로 두 오답을 설명하며 배제한다.
- 높은 이해도가 전투 정보 우위로 체감된다.
- 예측 성공 뒤에도 대응 선택이 남는다.
- 오답은 체력 손실뿐 아니라 새 판단 근거를 준다.
- 핵심 패턴 첫 발동이 불공정한 강제 피해로 느껴지지 않는다.
- 연구와 히든 분기가 최적해 파밍이 아니라 새로운 이야기로 느껴진다.
- 의뢰를 생략해도 핵심 사건의 일반 회수가 가능하다.
