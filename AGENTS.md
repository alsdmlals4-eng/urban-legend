# Urban Legend 작업 진입 규칙

현재 작업 기준은 [`[기획서]/00_프로젝트_허브/START_HERE.md`](<[기획서]/00_프로젝트_허브/START_HERE.md>)다. 과거 규칙 전문은 `[기획서]/[백업]/urban-legend/root/AGENTS.md`에 보존하며 기본 지시로 사용하지 않는다.

1. `ACTIVE_CONTEXT`와 `DOCUMENTATION_MAP`을 읽고 영향 분야의 본책·스킬 하나를 선택한다.
2. `[백업]`과 `[보류]`는 사용자가 재개를 요청할 때만 읽는다.
3. 서술형 변경은 해당 본책, 구조화 상태·경로·해시는 Registry/Manifest를 함께 갱신한다.
4. `project.godot`, 저장·캠페인·에피소드 데이터, 보호 코드·Scene은 명시적 기능 요청 없이 변경하지 않는다.
5. 완료 전에 보존 대조, 활성 링크, Registry, PDF/Manifest 및 해당 Godot 검증을 실행한다. 실행하지 않은 항목은 `NOT_RUN` 또는 `[미검증]`으로 기록한다.
