# 괴이 기록국 사운드 기획서

> 문서 위치: `[기획서]/04_사운드기획서.md` | 상태 기준: `codex/mvp048-campaign-novel`의 `4cf39a2` | 관련 표현 원칙: [`03_아트기획서.md`](03_아트기획서.md)

## 1. 사운드의 역할과 현재 상태

사운드는 “괴이가 나타났다”를 큰 소리로 선언하는 장치가 아니라, 기록된 현실과 현장의 미세한 어긋남을 느끼게 하는 정보·정서 레이어다. 플레이어가 소리를 끄거나 듣지 않아도 텍스트, 화자명, 색, 아이콘, 흔들림 없는 시각 변화만으로 같은 진행 정보를 이해할 수 있어야 한다.

현재 구현 확인된 사운드 요소는 기록관 아카의 절차적 접속 시그니처(`scripts/ui/log_guide.gd`)다. 사건별 BGM, 환경음, 선택 SFX, 보이스는 아직 기획·제작·QA가 필요한 영역이며 존재하지 않는 자산을 구현 완료로 기록하지 않는다.

## 2. 사운드 정체성

| 레이어 | 감각 | 사용 규칙 |
|---|---|---|
| 기록국/HQ | 낮은 환풍기, 종이·키보드·장비의 작은 물성 | 대사보다 앞서지 않는 저밀도 루프 |
| 현장 환경 | 역의 잔향, 빗물, 송신 잡음, 빈 공간의 반사 | 사건 규칙을 암시하되 정답 신호가 되지 않음 |
| 괴이 반응 | 미세한 역재생, 끊긴 안내, 방향이 어긋난 반향 | 위험 사례/전조에만 짧게, 반복 재생 금지 |
| 기록/검증 | 종이 넘김, 체크, 낮은 금속 점등 | 성공을 과장하지 않고 기록이 남았음을 알림 |
| 아카 | 짧은 절차적 모티프와 명료한 안내 | 최초 등장·중요 상태 변화만, 연속 대사마다 반복 금지 |

## 3. 장면별 큐 방향

| 장면 | BGM/환경 | SFX·보이스 규칙 | 무음 대체 |
|---|---|---|---|
| 도입·텍스트 노벨 | 절제된 HQ 또는 사건별 저밀도 패드 | 선택 포커스/결과 한 번, 화자 전환은 과장 금지 | 화자명·테두리·결과 텍스트 |
| 저승역 조사 | 형광등, 원거리 레일, 비정상 안내의 공백 | 책 페이지, 기록 비교, 위험 배너 | 페이지 탭·위험 표식·사례 제목 |
| 3×3/4×4 | 낮은 철도 드론, 연결 시 약한 황동 점등 | 회전, 연결, 고착, 도착을 분명히 구분 | 타일 점등·고착 아이콘·조작 수 |
| 회수/안정화 | 전조마다 공간감이 한 단계만 변함 | 자동 전조 1회, 선택 결과 1회 | 전조 제목·직전 결과·위험 사례 |
| HQ/외부 계약 | 사무 공간의 조용한 리듬 | 예약/취소/소비 결과를 서로 구분 | 비용·소비 상태·계약 ID |

## 4. 사건별 팔레트

- **저승역:** 역의 먼 저주파, 목적지 공백 뒤의 비정상적인 잔향, 정식 식별음의 안정된 짧은 종결. 개인 목적지나 공포음을 정답처럼 사용하지 않는다.
- **빨간 우산:** 비, 젖은 골목, 멀어지는 보행 소리. 위험은 음량 폭발 대신 거리와 반사의 변화로 표현한다.
- **폐주파수 방송국:** 통신 잡음, 규칙적이지만 출처가 다른 펄스. 정보가 들리지 않으면 텍스트 기록만으로도 비교 가능해야 한다.
- **새벽 2시 17분의 도착 알림:** 진동 알림의 잔상, 시간표의 기계적 틱, 알림이 지나간 뒤의 정적. 특정 음 높이나 패턴을 외워야만 풀 수 있게 만들지 않는다.

## 5. 접근성·믹스·제작 기준

- BGM/SFX/보이스/시스템 볼륨을 분리하고, 음소거에서도 필수 정보가 손실되지 않게 한다.
- 공포성 자극은 반복 횟수·최대 음량을 제한하고, 화면 흔들림·섬광과 동시에 과도하게 겹치지 않는다.
- 보이스가 도입되더라도 스크립트, 화자명, 대사 로그를 항상 제공한다. 성우 더빙은 플레이 진행을 막는 필수 수단이 아니다.
- 파일명은 `scene_event_variant`처럼 장면·행동·변형을 알 수 있게 하고, 출처·라이선스·루프 길이·정규화 값·무음 대체를 manifest에 기록한다.

## 6. 제작 로드맵과 QA

1. 아카 시그니처의 재생 빈도·음소거 동등성을 먼저 QA한다.
2. 저승역의 조사/보드/회수에 환경 루프와 핵심 SFX 소량을 배치한다.
3. 나머지 세 사건은 동일한 큐표와 무음 대체 계약으로 확장한다.
4. BGM/SFX/보이스 슬라이더, 헤드폰/스피커, 음소거, 연속 재진입을 QA한다.

사운드를 추가·변경하면 이 문서, asset manifest, [QA 기획서](05_QA기획서.md)를 함께 갱신한다. 시각과 동시 변경되는 연출은 [아트 기획서](03_아트기획서.md)와도 심사한다.
