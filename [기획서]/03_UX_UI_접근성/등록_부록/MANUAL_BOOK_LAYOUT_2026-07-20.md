# 책형 이상 매뉴얼 화면 구조 — 2026-07-20

관련 Issue: #44

## 확정 구조

모든 대상 화면은 아래의 역할 분담을 유지한다.

- 좌측: 현재 위치를 식별할 수 있는 현장 이미지. 회수 페이즈에서는 대표 요원과 팀 상태를 함께 표시한다.
- 중앙 상단·중단: 현재 상황 지문과 실제 조작/연출 영역.
- 중앙 하단: 다음 진행 버튼, 미니게임 결과, 또는 회수 대응 선택지.
- 우측: 항상 열린 책형 `이상 매뉴얼`. 내용만 스크롤하며 게임 입력이나 판정 규칙을 소유하지 않는다.

## 적용 범위

| 화면 | 중앙 조작 | 책에 표시하는 정보 |
| --- | --- | --- |
| 저승역 긴급 보고 | 현재 지문과 대화 선택지 | 현재 규칙, 단서, 기록원 지원, 투입 요원 |
| 노선 복원/일반 미니게임 | 실제 미니게임 보드와 결과 | 조작 규칙, 비교 근거, 현장 기록 |
| 회수 페이즈 | 전조, 괴이 연출, 세 개의 대응 카드 | 자동 반영, 예측/단서, 오대응 학습, 요원 지원 |

## 제외 및 보호

- `GameState`, 저장 형식, 에피소드 데이터, 회수 판정 공식은 변경하지 않는다.
- 회수는 괴이를 처치하는 전투가 아니라 전조와 단서를 대조해 안정화·회수하는 판단 단계다.
- 오른쪽 책은 표시 전용이다. 선택지와 미니게임 입력은 중앙에 남긴다.

## 1280×720 기준

오른쪽 책은 접히지 않는다. 좁은 폭에서는 책 안의 정보만 스크롤하고, 중앙의 세 대응 카드는 동일한 한 행을 유지한다.
