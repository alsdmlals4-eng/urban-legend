# Base → Urban Legend 운영체계 이주 인터뷰

## 확정 계약

Base `eb40b912e5f5a0e4d369105a4f0a770e0a6179a9`의 필요한 운영 기능만 Urban Legend에 프로젝트 전용으로 분화한다. 작업은 보존 감사, 거버넌스·스킬 동기화, 발행·Health Review의 세 PR로 나눈다.

## 보호 대상

`project.godot`, `scripts/core/game_state.gd`, `data/episodes/**`, 기존 Scene, 승인 자산, 실제 QA 결과와 현행 책임 원본은 변경하지 않는다. 게임 기능 불일치는 후속 Issue로만 분리한다.

## 제외 범위

Base 전체 파일·사례·스킬의 복사, 게임 기능·저장 스키마 변경, 승인 없는 삭제·대규모 이동은 제외한다.
