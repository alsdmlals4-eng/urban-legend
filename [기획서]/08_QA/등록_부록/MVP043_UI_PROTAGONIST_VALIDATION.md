# MVP-043 현장 UI·회수 루프·5인·캐릭터 아트 검증

> 검증일: 2026-07-16
>
> 기준: `docs/CODEX_GOAL_MVP_043.md`, 커밋 `c17d58c`, Issue #36
>
> 범위: 중앙 조사 진행, 팀 상태·기록관 아카, 자동 전조 회수, 권나래 일정 대시보드, 초기 5인 저장 호환, 캐릭터 아트 28종, 기존 사건 회귀

## 구현 확인

- 저승역 조사 화면을 최대 1600px 중앙 안전 프레임과 3열 구조로 교정했다.
- 기존 F2 자유 배치 설정이 핵심 조사 구조를 재부모화하거나 이동하지 않도록 제외했다.
- 선택 가능한 지점 또는 후보가 없어도 다른 조사 지점 복귀와 HQ 복귀 경로를 유지한다.
- 성공·실패 사례와 5개 후보를 표시 카탈로그로 제공하고 보호 사건 JSON 구조는 변경하지 않았다.
- 비활성 후보는 제거 이유를 표시하며, 위험 행동 뒤에는 같은 목록으로 복귀한다.
- 권나래를 모든 `selected_agent_ids`의 첫 위치로 승격하고, 현재 반일의 기존 주인공 일정만 권나래에게 복사한다. 원본 일정과 진행 중 작전은 보존한다.
- 윤서하·한유리를 초기 편성 후보에 추가하고, 박도윤·이세린·레이먼드 케인·카밀라 바르가스는 외부 접점 카드로만 제공한다.
- 반일 일정은 주인공 한 명만 필요·소비하며 서포트는 독립 일정을 요구하지 않는다.
- 자동 지원 시도는 성공·실패 모두 기존 `used_agent_supports`에 남아 재진입 재추첨을 막는다.
- 회수는 전조를 자동 관측하고 행동 직후 `직전 결과`와 다음 전조를 동시에 표시한다. 대응 카드는 가로 3열, 소비품은 별도 보조 행이다.
- 플레이어 노출 `LOG`·`로그`는 `아카 안내`·`기록관 아카 · 괴이 기록국 관제 AI`로 치환한다. 내부 ID와 저장 키는 유지한다.
- 초기 5인 20종과 외부 접점 4인 8종을 독립 PNG로 제작하고, 원본 카드와 생성 원본은 `.gdignore` 경로에 보존했다.
- 노선 복원은 3×3 학습과 4×4 최종 검증을 한 컨트롤 안에서 구분하며 최종 단계만 결과를 저장한다.
- 완료 결과와 기록국 DB는 기존 미니게임 결과 상세값으로 안전 노선 기록과 매뉴얼 페이지를 표시한다.

## 실제 장면 검수

- 1280×720 및 1920×1061 실제 OpenGL 창에서 조사, 위험 결과, 팀 상태, 회수 반복, 일정 대시보드, 외부 접점을 캡처했다. 1920 요청은 현재 Windows 작업 영역 때문에 세로 1061px로 생성됐다.
- 조사 기본 화면에서 상단 HUD, 세 열, 첫 후보, 복귀 버튼이 화면 안에 있고 한국어 본문과 제거 근거가 내부 스크롤로 유지되는 것을 확인했다.
- 팀 상태 팝오버가 화면 밖으로 밀리는 문제를 캡처에서 발견해 중앙 정렬로 수정·재캡처했다.
- 일정 화면의 행동 카드를 2×2로 압축하고 하단 탭을 화면 안에 유지했다. 외부 접점은 하단 탭 내부 스크롤에서 열람한다.
- 캡처 원본은 `docs/qa/captures/mvp043/integrated/`에 보존한다.

## 미검증·남은 위험

- 실제 장시간 마우스·키보드 플레이와 1918×943 전용 창 캡처는 미검증이다.
- 첫 10분의 실제 소요 시간과 플레이어가 사례 비교 원칙을 이해하는지는 사용자 플레이테스트가 필요하다.
- 다른 사건 콘텐츠는 재작성하지 않았으며 공용 흐름 smoke만 확인했다.
- GDD DOCX는 생성 해시·문단 224개·표 34개·이미지 5개 구조 검사를 통과했다. 현재 실행 환경에 LibreOffice/Word 변환기가 없어 페이지 PNG 렌더 검수는 미검증이다.

## 저장 경계

- 새 저장 필드, 저장 버전, 마이그레이션 없음.
- 보호 사건 JSON 구조 변경 없음.
- 주인공·서포트는 기존 `selected_agent_ids`, 지원 시도는 기존 `used_agent_supports`를 재사용한다.

## 사용자 확인 순서

1. 신규 캠페인에서 저승역 조사 화면에 진입한다.
2. 조사 장소를 바꾸고 플랫폼 스피커의 5개 후보와 페이지형 매뉴얼을 확인한다.
3. 실패사례를 열어 오답 비활성화 이유와 남은 후보 수를 확인한다.
4. 위험 후보를 선택해 위험 사례가 남고 같은 목록으로 복귀하는지 확인한다.
5. 3×3 학습 뒤 4×4 최종 검증과 결과·DB 매뉴얼을 확인한다.
6. HQ에서 권나래가 고정 주인공인지, 윤서하·오현·강이준·한유리 중 서포트를 최대 둘 고를 수 있는지 확인한다.
7. 회수에서 행동 직후 직전 결과와 다음 전조가 함께 남는지 확인한다.
8. 외부 접점 탭에서 네 NPC 카드를 확인하되 편성·고용 버튼이 없는지 확인한다.
