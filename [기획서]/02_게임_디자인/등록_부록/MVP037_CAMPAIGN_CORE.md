# MVP-037 10일 비선형 데모 캠페인 코어

## 목표와 플레이 시간

- 필수 사건 진행: 약 125~155분
- 일상·연구·세력 콘텐츠 포함: 약 180~220분
- 최대 10일 안에 세 사건을 원하는 순서로 조사한다.

## 이번 구현 범위

- `CampaignState`가 일차, 오전·오후 일정, 사건 공개 상태, 위험도 순환, 긴급 사건, 일상 이해도와 데모 종료를 계산한다.
- `GameState`는 해당 계산을 저장·불러오기와 화면에 연결한다.
- 준비 화면에서 전 요원의 오전·오후 일정을 정해야 조사를 시작할 수 있다.
- 결과 화면에서 사건 준비로 돌아갈 때 활성 작전의 하루를 한 번만 마감한다.
- 저장 스키마는 `mvp-037`이며 캠페인 필드가 없는 `mvp-035` 저장에는 안전한 초기값을 적용한다.

## 고정 규칙

- 저승역과 빨간 우산은 `lead`, 폐주파수 방송국은 `unknown`으로 시작한다.
- 하루 위험 상승 대상은 기본 1건, 확산이 높은 날만 2건이다.
- 순환 커서를 사용해 미해결 사건이 2일 넘게 제외되지 않게 한다.
- 위험 60에서 자동 발견하고 100에서 긴급 사건으로 지정한다. 폭주는 게임 오버가 아니다.
- 일상 이해도는 하루 한 번만 받고 사건별 최대 +10이다. 최초 콘텐츠는 +2, 후속은 +1이다.
- 10일차는 플레이 가능하며 다음 하루 진행 시 데모 종료 상태가 된다.

## 아직 구현하지 않은 후속 범위

- 폐주파수 방송국의 실제 에피소드 데이터·장면·미니게임
- 일상 에피소드 선택 화면과 실제 대사 콘텐츠
- 해결 등급별 연구점 6/10/14, 연구 위험 모드와 연구 트리
- 전역 패시브, 설계도, 잔향 파편 제작, 개인 2슬롯 장비와 계통 제한
- 용병·세력 파견, 10일 종료 보고서와 미해결 사건 목록

이 항목들은 현재 캠페인 저장 구조를 재사용하되, 콘텐츠와 경제 규칙별 실패 테스트를 먼저 추가한 뒤 구현한다.
